发送短信验证码

接口描述

  • 发送短信验证码

URL

  • {URL}/common/get_sms

请求方式

  • GET

请求参数

参数名 必选 类型 说明
mobile 是 string 手机号
captcha_key 是 string 图片验证码key
captcha_value 是 string 图片验证码

请求示例

GET /common/get_sms?mobile=18601032623&captcha_key=Qyx10L5K5iESftzMVewUz3YkjqmOMZXM&captcha_value=wqk4 HTTP/1.1

返回参数

参数名 类型 说明
status number 返回状态编号，200为正常返回，其它值为错误编码
message string 返回状态信息
data object
- sms_status
 number 短信发送状态，枚举值：[0-发送成功，-1-账号未注册，-2-其他错误，-3-密码错误，-4-手机号格式不对，-5-余额不足，-6-发送时间格式不正确，-7-短信内容未加签名，-101-调用接口速度太快]

返回示例

{
  "status": 200,
  "message": "加载完成",
  "data": {
    "sms_status": 0
  }
}
